Scatter Plot Matrix (SPLOM)

Data: Challenger data (from Moodle)

Features:
- Mouseover highlights all corresponding points by increasing size
- Clicking on a point colors all corresponding points red
- Clicking again changes the points back to black
- Tooltips for the points give the flight index


Parallel Coordinates:

Data: Forest fire data
https://archive.ics.uci.edu/ml/machine-learning-databases/forest-fires/

Features:
- Mouseover highlights the selected line
- Tooltips for the lines give the months of the fire
